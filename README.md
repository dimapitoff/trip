# trip

Статический сайт с маршрутом поездки.

## Открыть сайт

После публикации на GitHub Pages сайт доступен по ссылке:

https://dimapitoff.github.io/trip/

## Как обновлять сайт

1. Внесите изменения в файлы проекта.
2. Сделайте commit и push в ветку `main`.
3. GitHub Actions автоматически задеплоит новую версию.

Проверка статуса деплоя: вкладка Actions в репозитории.

## API-ключ Яндекс Карт

Для работы своей карты используется API-ключ Яндекс Карт.

1. Откройте файл `config.local.js`.
2. Укажите ключ:

```js
window.YANDEX_MAPS_API_KEY = "ВАШ_API_КЛЮЧ";
```

3. Перезагрузите страницу.
